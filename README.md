# 💍 Convite Digital Interativo

Projeto desenvolvido para criar uma experiência digital moderna e personalizada para um casamento, substituindo convites tradicionais por uma solução elegante, responsiva e interativa.

## ✨ Objetivo

Criar um convite digital visualmente agradável, funcional e acessível, permitindo que convidados tenham acesso rápido às informações do evento de forma prática e moderna.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🎨 Funcionalidades

- Layout moderno e responsivo
- Design elegante e minimalista
- Navegação suave
- Informações do evento organizadas
- Interface adaptada para dispositivos móveis
- Experiência visual personalizada

---

## 📱 Responsividade

O projeto foi desenvolvido com foco em responsividade, garantindo uma boa experiência tanto em computadores quanto em smartphones.
